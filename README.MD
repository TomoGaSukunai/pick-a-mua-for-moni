# Pick a MUA

A toy to get member list of a live show room on bilibili  
Then pick a lucky one 

## Devlop

Nodejs required.  
Better use yarn as packages manager, although npm is OK.

### Clone
```bash
git clone <repo> [dir] 
cd dir
```

### Dependencies Install
```bash
yarn
```

### Debug/Run 
```bash
yarn start
```

## Build
```bash
yarn dist
```
